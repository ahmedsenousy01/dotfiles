## Hotkeys

- **Left Half**: `Cmd+Option+Left`
- **Right Half**: `Cmd+Option+Right`
- **Top Half**: `Cmd+Option+Up`
- **Bottom Half**: `Cmd+Option+Down`
- **Maximize**: `Cmd+Option+Enter`
- **Center**: `Cmd+Option+C`
- **Restore**: `Cmd+Option+Backspace`
